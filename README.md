## About

This is the portfolio website for **Nikholas Novachuk**, an Orthodox wedding photographer based in Arizona. The site showcases candid, documentary-style wedding photography with a focus on capturing authentic moments during Orthodox Christian ceremonies.

## Tech Stack

- **Framework:** React 19
- **Language:** TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Deployment:** Vercel

## Features

- Responsive design optimized for all devices
- Smooth scroll animations with Framer Motion
- Image optimization with vite-plugin-image-optimizer
- Gallery embedding system for showcasing wedding collections
- Contact form integration
- FAQ section with expandable answers

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |
| `npm run lint` | Run ESLint |
| `npm run add-gallery` | Add a new gallery to the site |
| `npm run deploy` | Build and deploy to production |
